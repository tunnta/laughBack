# コンセプト
シンプルで直感的なデザインの大喜利アプリです
# 目的
大喜利を通したコミュニケーションを誰もが楽しめるように、アクセスのしやすさや直感的な使用感を重視しました。
# 機能
snsユーザー認証(Auth0)  
お題の投稿、お題に対する回答の投稿  
回答への'いいね'機能  
3日経過したお題をお題に付随する、各回答のいいねの数を集計、ソートし、top画面の現在のお題一覧の下に移動し表示。  
ランキング。  
PCとスマホで画面の異なる、レスポンシブ仕様。  
# 技術
HTML/CSS  
javascript  
Next.js  
Ruby on Rails 6.1.4.1  
Ruby 2.6.7  
Auth0  
Heroku  
Vercel
